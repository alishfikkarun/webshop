
Как установить:

1. Создать базу данных
2. Выполнить команду для папки проекта npm i
3. Выполнить команду для папки проекта npm run firstInit это создаст структуру базы, установить колонки и создаст пользователя admin с паролем admin для входа
4. Перейти в папку server и выполнить настройку .env согласно примеру .env.example
5. Перейти в папку client и выполнить настройку .env согласно примеру .env.example
6. Запустить выполнение проекта npm start
7. После запуска откроется страница самого магазина, для перехода в панель необходимо перейти по адреса /admin и ввести логин и пароль admin
